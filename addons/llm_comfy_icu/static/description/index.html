What is ComfyICU?

ComfyUI as a cloud service

ComfyICU provides ComfyUI workflows as a cloud API service. This module integrates Odoo with the ComfyICU API, allowing you to generate images and media without running your own ComfyUI infrastructure. Just connect and generate.

Why ComfyICU?

Benefits of cloud-based media generation

No Infrastructure

No need to manage GPU servers or ComfyUI installations. The cloud handles everything.

Quick Setup

Get started quickly with just an API key. No complex setup or hardware requirements.

Scalable

Generate as many images as you need. Cloud resources scale automatically.
